# DOE NETL Energy Data Exchange (edx-netl-doe-gov)

DOE NETL Energy Data Exchange is a federal government open-data portal for United States running **CKAN**, exposing a standardized catalog API over ~17,982 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/edx-netl-doe-gov/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=edx-netl-doe-gov-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 3.0.0

## API
- **DOE NETL Energy Data Exchange CKAN Action API** — base `https://edx.netl.doe.gov/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://edx.netl.doe.gov)
- Live endpoint verified: `https://edx.netl.doe.gov/api/3/action/status_show` · ~17,982 datasets
